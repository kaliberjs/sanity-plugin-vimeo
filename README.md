# Kaliber Sanity Vimeo Asset Source

Select video's from Vimeo

## Installation

```
> yarn add @kaliber/sanity-plugin-vimeo-asset-source
```

`admin/sanity.json`

```json
{
  "plugins": [
    "@kaliber/sanity-plugin-vimeo-asset-source",
    ...
  ],
  ...
}
```
